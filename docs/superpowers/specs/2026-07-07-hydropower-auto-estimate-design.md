# 水电自动点位与管线估算设计

## 背景

当前商品房整装默认报价中，强电布线、弱电布线和水路布管按项目级建筑面积计价。这种方式实现简单，但给客户的解释力不足：客户看不到开关、插座、灯位、冷热水点、排水点、线管和水管这些可理解的明细，容易觉得价格笼统。

新的目标是不新增必填 CAD 图层，仍尽量依赖现有空间识别结果，自动生成水电点位候选和管线估算量；报价员或设计师在前端复核数量后，导出更可信的客户报价。

## 目标

- 水电报价从“建筑面积 * 单价”改为“点位数量 + 管线延米”的客户可解释清单。
- 第一版不要求设计师新增水电 CAD 图层。
- 系统按空间类型、楼层、厨卫数量和已有洁具信息自动生成带坐标的虚拟点位候选。
- 厨房、卫生间、阳台、洗衣房等水电密集空间给出更细默认规则。
- 别墅和复式按楼层、空间类型和空间数量自动叠加，支持更多空间类型。
- 管线长度优先基于虚拟点位坐标估算，并明确标记为系统推荐估算，不伪装成 CAD 实测。
- 前端提供水电点位复核入口，允许报价员调整数量和启用/关闭可选点位。

## 非目标

- 第一版不做真实水电施工图解析。
- 第一版不强制设计师画开关、插座、灯位、给排水、线管或水管图层。
- 第一版不计算真实走线绕梁、绕墙、走顶或走地路径。
- 第一版不把系统默认候选当成已确认工程量；未复核时需要在报价风险中提示。

## 核心模型

新增一个项目级水电估算结果，按空间生成明细，再汇总到 Excel 水电工程：

- 空间级候选：每个空间包含自动生成的带坐标点位和估算管线。
- 项目级汇总：把同类点位和管线汇总为报价项。
- 复核状态：记录水电点位是否已由报价员确认。

建议的内部字段：

- `hydropower_points`: 空间级点位数组，每个点位包含类型、数量、坐标、来源和复核状态。
- `hydropower_pipe_lengths`: 空间级管线估算数组。
- `hydropower_review_status`: `auto_estimated`、`confirmed` 或 `needs_review`。
- `hydropower_summary`: 项目级汇总，用于报价映射和 Excel。

点位精度分三层：

1. 真实点位：后续如果 CAD 中出现真实水电点位图块，系统直接使用真实坐标。
2. 系统推荐点位：第一版主路径。系统根据空间轮廓、墙线、门窗、柜体、洁具、空间中心和楼层信息生成虚拟坐标。
3. 数量兜底：当空间轮廓或关键几何不足时，只生成数量候选，并在管线估算和健康检查中标记为低置信度。

第一版以系统推荐点位为主，不要求真实水电图层。推荐点位需要在图形校对页中可视化，报价员可以看到这些点位落在哪面墙、哪个空间或哪个洁具附近。

点位类型第一版建议包含：

- 电路：开关点位、普通插座、沙发充电插座、取暖设备插座、床尾风扇插座、厨房台面插座、灯位、弱电点位、空调专线、大功率电器专线、浴霸/暖风机专线、智能马桶插座、洗衣机插座、烘干机插座候选、净水机插座。
- 水路：冷水点、热水点、排水点、地漏、马桶给水点、浴室柜冷热水点、浴室柜排水点、花洒冷热水点、厨房水槽冷热水点、厨房水槽排水点、洗衣机给水点、洗衣机排水点、净水机给水点。
- 管线：强电线管、弱电线管、给水管、排水管。

## 默认点位规则

默认规则按空间类型生成。数量是候选值，报价员可以调整。

默认规则不仅生成数量，也生成推荐位置。推荐位置使用现有图形数据：

- 空间闭合轮廓用于确定房间边界、中心点和可放置墙边。
- 门洞用于推断入口和开关位置。
- 窗洞用于避让插座、电视点位和大功率点位。
- 厨房地柜/吊柜线用于推断台面插座、净水机、水槽和电器点位的墙边。
- 马桶和浴室柜点位优先使用现有 `QUOTE_TOILET` 和 `QUOTE_BATHROOM_VANITY`。
- 如果没有洁具点位，则按卫生间墙边和远离门洞原则生成推荐点。

## 别墅和复式处理

别墅的关键不是使用另一套公式，而是让默认规则按楼层和空间数量自然叠加，同时支持更多空间类型。

- 所有点位按空间实例生成，不按空间名称去重。例如二层两个卧室，各自生成卧室点位。
- Excel 汇总可以按项目级同类项合并，备注中保留来源空间数量。
- 前端复核面板按楼层分组展示，便于别墅逐层检查。
- 多楼层楼梯空间生成照明和开关候选，但避免每层重复生成过多插座。
- 茶室、娱乐室、健身房、影音室按干区功能房处理，默认强电和弱电点位不低于书房。
- 露台、设备平台、洗衣房、保姆房、储藏间等空间按已有分类或关键词扩展规则生成候选。
- 挑空空间不直接按普通房间生成完整点位，只生成照明和开关候选，避免因无地面使用功能而高估。
- 地下室、负一层、负二层按正常楼层处理；如果空间类型为娱乐室、储物间、设备间，使用对应空间规则。

## 管线估算

没有真实水电图层时，管线不能当作 CAD 实测，但可以基于系统推荐点位坐标做按图估算。估算应稳定、可解释，并在 UI 和报价备注中标记。

管线估算分三层：

1. 有真实管线：后续如果 CAD 有真实管线图层，直接量取线段延米。
2. 有推荐点位：第一版主路径。按点位坐标到空间入口、墙边、湿区主管候选点和楼层干线的距离估算。
3. 只有数量：当点位没有坐标时，退回点位数量乘经验系数，并标记为低置信度。

建议第一版公式：

- 强电线管 = 推荐强电点位到空间入口/墙边干线的估算距离合计 + 专线额外长度 + 每层强电基础干线。
- 弱电线管 = 推荐弱电点位到空间入口/墙边干线的估算距离合计 + 每层弱电基础干线。
- 给水管 = 冷热水点到湿区主管候选点的估算距离合计 + 厨卫湿区基础管长。
- 排水管 = 排水点和地漏到湿区排水主管候选点的估算距离合计 + 厨卫阳台基础排水管长。
- 如果空间缺少有效推荐点位坐标，则退回数量系数公式。

第一版默认系数应放在可配置规则中，便于后续按报价习惯调整。示例：

- 强电点位支管系数 2.5m。
- 弱电点位支管系数 2.5m。
- 专线支管系数 6m。
- 给水点支管系数 1.8m。
- 排水点支管系数 1.2m。
- 厨房给水基础管长 3m。
- 卫生间给水基础管长 4m。
- 每层强电基础干线 8m。
- 每层弱电基础干线 6m。

如果后续 CAD 增加配电箱、弱电箱、管井或厨卫主管点位，可用空间中心点距离修正基础干线；第一版不依赖这些数据。

距离计算规则：

- 点位到墙边干线优先使用点位到最近墙线或空间边界的投影距离。
- 空间入口优先取门洞中点；没有门洞时取空间边界上靠近楼层中心的一点。
- 湿区主管候选点优先取厨房水槽、浴室柜、马桶或洗衣机点位附近的聚类中心。
- 估算长度应乘以余量系数，例如 1.15，用于覆盖绕线、上下翻和施工损耗。
- 每条估算记录保留来源说明，例如 `virtual_point_distance`、`fallback_count_factor`。

## 前端复核

新增水电点位复核面板，建议放在 Excel 可选补项附近或报价映射面板之前。

功能：

- 按楼层和空间展示点位候选。
- 在图形校对页显示“水电推荐点位”图层。
- 每个点位数量可编辑。
- 点位坐标第一版至少可查看；拖动点位可作为第二阶段能力。
- 厨房、卫生间、阳台提供常见可选项开关，例如智能马桶、净水机、洗衣机、烘干机、大功率电器专线。
- 显示项目级汇总。
- 显示管线估算公式和估算结果。
- 用户点击“确认水电点位”后，报价映射和 Excel 使用确认值。
- 校对快照保存复核后的点位、管线和确认状态。

## 健康检查

新增水电相关提示：

- 如果使用默认候选但未确认，显示 info：水电点位为自动估算，建议导出前复核。
- 如果点位只有数量没有坐标，显示 warning：水电管线只能按数量系数估算，建议检查空间轮廓。
- 如果厨房、卫生间、阳台、洗衣房缺少关键水电点位，显示 warning 或 info。
- 如果水电规则仍使用 `building_area_m2`，显示 warning：水电仍按建筑面积计价，不符合当前客户报价口径。

## 数据流

1. 后端解析 DXF 并输出空间、楼层、空间类型、洁具数量等现有字段。
2. 前端或共享纯函数根据空间行和图形几何生成带坐标的水电推荐点位。
3. 前端复核面板和图形校对页展示推荐点位，并允许调整候选数量。
4. 管线估算函数基于推荐点位坐标生成强电、弱电、给水和排水延米。
5. 报价映射读取复核后的候选；没有复核时读取默认候选并标记风险。
6. Excel 水电工程按项目级水电 metric 汇总输出。
7. 校对快照保存点位坐标、复核结果、管线估算和确认状态，导入后恢复。

第一版建议把水电候选生成放在前端纯函数中，便于快速迭代和测试；如果后续需要 API 直接输出水电候选，再把规则迁移到后端或共享规则文档。

## 测试

- 纯函数测试：不同空间类型生成正确默认点位。
- 点位坐标测试：厨房、卫生间、卧室等空间生成的点位落在对应空间轮廓内或墙边容差范围内。
- 别墅测试：多楼层、多同类空间按空间实例累加，不误去重。
- 厨房测试：生成台面插座、净水机、水槽冷热水、排水和大功率专线。
- 卫生间测试：结合 `toilet_count` 和 `bathroom_vanity_count` 生成马桶、浴室柜、花洒和地漏点位。
- 阳台/洗衣房测试：生成洗衣机给排水和插座。
- 管线估算测试：点位数量变化后线管和水管延米同步变化。
- 管线坐标估算测试：移动或改变推荐点位后，估算长度变化方向合理。
- 图形数据降级测试：空间缺少有效几何时退回数量系数，并输出低置信度提示。
- 报价映射测试：水电新 metric 进入 Excel，旧建筑面积水电规则被替换。
- 快照测试：复核后的水电点位和确认状态可保存恢复。

## 迁移策略

- 保留 `electrical_scope_area_m2` 和 `plumbing_scope_area_m2` 作为旧字段兼容，但默认规则不再使用它们。
- 保留自定义规则导入能力；如果用户导入旧规则，健康检查提示水电仍按面积。
- 默认报价规则 JSON 更新为新水电 metric。
- 文档同步说明：水电点位为系统推荐点位，管线为基于推荐点位的按图估算，不是 CAD 实测。
