# 水电报价大类汇总输出设计

## 背景

当前主线已经合并了水电点位与管线估算：系统可以根据空间类型、空间轮廓、洁具、柜体等信息生成细分点位和管线长度，例如厨房台面插座、智能马桶插座、空调专线、强电线管、给水管、排水点等。

这些细分数据适合复核，但不适合直接进入客户最终报价表。客户报价表需要更清爽的工程分类，便于核价、打印、解释和后续修改单价。

因此本设计新增一层“水电报价聚合层”：底层保留细分来源，最终报价表只输出水电汇总大类。

## 目标

- 最终 Excel 报价表的“水电工程”只输出汇总大类，不输出各空间水电明细。
- 报价规则单价表维护汇总大类单价，方便设计师统一改价。
- 水电复核面板保留空间级和细分来源，方便核对每个点位来自哪里。
- 商品房、复式楼、别墅使用同一套汇总口径。
- 避免点位、线管、水管之间漏项或重复计价。

## 非目标

- 不在最终报价表中输出按房间展开的水电明细。
- 不新增灯带报价项，当前无法稳定判断。
- 不拆分筒灯和射灯，统一为“筒灯/射灯”。
- 不单列空调专线，空调和其它设备专线统一进入“设备专线”。
- 不新增回水管、户外给水、户外排水。
- 不把复核面板做成独立水电报价单页面；复核面板只是数据来源核对入口。

## 数据流

1. `buildHydropowerEstimate` 继续生成细分点位、空间归属、估算管线和项目级细分汇总。
2. 新增水电聚合函数，把细分汇总和复核面板手填项合并为报价大类汇总。
3. `buildQuoteMapping` 使用水电大类汇总生成报价项。
4. `quote-excel` 的“水电工程”只输出大类汇总行。
5. 水电复核面板仍显示空间级细分来源，可调整细分点位和手填项。
6. 校对快照保存细分点位、管线复核结果和手填大类数量。

## 最终报价表输出规则

- 最终报价表只保留“水电工程”汇总大类行。
- 不输出水电按房间明细。
- 数量为 0 的水电大类不输出，除非是需要设计师填写的固定占位项。
- 强电箱、弱电箱默认数量可为 1，分配电箱默认 0。
- 筒灯/射灯默认 0，只有复核面板录入后才进入最终报价。
- 总价、小计、管理费和税金沿用现有 Excel 逻辑。

## 复核面板规则

复核面板需要同时服务两件事：

- 给设计师看清楚系统如何得到大类数量。
- 允许设计师修改自动估算不可靠的数量。

建议复核面板分三层：

1. 顶部项目级大类汇总：强电插座、开关、灯位、设备专线、给排水点位、管线等。
2. 空间级明细：每个空间显示细分点位来源和管线估算。
3. 手填/可选项：筒灯/射灯、强电箱、弱电箱、分配电箱、额外设备专线。

复核面板的空间级明细不会进入最终报价表，只作为数据依据和校对入口。

## 风险和处理

- 点位和管线可能被客户理解为重复计价：工艺说明要说明点位是底盒、开槽、接线等服务，管线是线管/水管和敷设延米。
- 管线长度是估算值，不是真实施工图实测：报价说明保留“按系统估算，设计师可复核调整”。
- 筒灯/射灯缺少可靠图层或识别依据：第一版只允许手填，不自动推断。
- 复式楼和别墅的强弱电箱数量可能多于 1：强电箱、弱电箱、分配电箱必须可改。
- 旧报价规则 JSON 可能仍含细分水电规则：导入旧规则时需要补齐新大类规则，默认导出使用新大类。

## 验收标准

- 默认报价规则单价表显示水电大类规则，而不是一堆细分插座规则。
- 下载报价规则 JSON 包含水电大类规则。
- 导出 Excel 草稿的“水电工程”只显示大类汇总行。
- Excel 草稿不输出水电空间明细。
- 水电复核面板仍能看到细分来源，并能调整筒灯/射灯、强电箱、弱电箱、分配电箱等数量。
- 商品房、复式楼、别墅都使用同一套水电大类汇总逻辑。
- 现有非水电报价规则和空间工程输出不受影响。
