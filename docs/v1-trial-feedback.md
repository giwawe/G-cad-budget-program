# V1 试用反馈清单

更新时间：2026-07-13

本清单用于 V1 小范围真实试用。目标是让设计师和报价员用 1-2 个真实项目跑完整流程，记录阻塞点和容易误解的地方，再决定 V1.1 做哪些小改进。

当前阶段只收集和分级反馈，不直接扩大到登录、DWG、后台管理、合同等平台化功能。

## 试用范围

每个试用项目建议至少走完一次完整流程：

1. 上传规范 DXF。
2. 查看方案完整性复核。
3. 查看健康检查和 CAD 修图清单。
4. 校对空间工程量。
5. 复核水电点位。
6. 填写报价抬头。
7. 导出 Excel 草稿。
8. 打开 Excel 检查打印预览、公式联动和关键大项。
9. 导出校对快照。

## 反馈分级

| 等级 | 含义 | 处理建议 |
| --- | --- | --- |
| 必须修 | 影响解析、金额、Excel 导出、公式联动或正式报价可信度 | 进入 V1.1 优先修复 |
| 近期优化 | 不影响金额，但影响设计师效率、理解成本或校对体验 | 进入 V1.1 或 V1.2 |
| 后续版本 | 平台化、大前端改版或新业务模块 | 进入后续规划，不阻塞 V1 试用 |

## 环节检查表

### 上传和解析

- [ ] DXF 上传后能正常返回结果。
- [ ] 空间数量和设计方案一致。
- [ ] 建筑面积、地面面积没有明显偏差。
- [ ] 没有无法解释的解析失败或空白结果。

### 方案完整性复核

- [ ] 房间边界显示清楚。
- [ ] 墙线长度能帮助判断墙体是否漏画。
- [ ] 门窗位置和空间归属符合预期。
- [ ] 更多图层中的外墙、洞口、洁具、水电推荐点位可按需查看。

### 健康检查

- [ ] warning 能说明需要修 CAD 的具体原因。
- [ ] info 不会误导设计师以为必须修图。
- [ ] CAD 修图清单可直接发给画图人员。

### 空间工程量校对

- [ ] 空间类型可理解，必要时能手动调整。
- [ ] 地面面积、墙线长度、窗洞面积、墙面乳胶漆等关键数值可快速核对。
- [ ] 厨房、卫生间、阳台、露台等湿区的墙砖和防水结果符合预期。
- [ ] 不计价空间能标为“不计价”。

### 水电点位复核

- [ ] 强电插座、开关、灯位数量能被设计师理解。
- [ ] 给水点、热水点、排水点数量能被设计师理解。
- [ ] 设计师知道这些点位是系统推荐，不是真实水电施工图实测。
- [ ] 调整后导出的 Excel 数量同步变化。

### Excel 导出

- [ ] 硬装 Excel 能导出并打开。
- [ ] 整装 Excel 能导出并打开。
- [ ] 硬装 + 自选增项 Excel 能导出并打开。
- [ ] 地址名称、客户、设计师、报价员、报价日期正确进入表格。
- [ ] 强弱电工程、给排水工程、主材、定制、室内门、卫浴、吊顶等关键大项没有漏项。
- [ ] 修改数量或三段单价后，小计、直接费合计和总价能自动联动。
- [ ] A4 打印预览可读，文字没有明显遮挡。

### 快照恢复

- [ ] 导出校对快照成功。
- [ ] 重新导入快照后，空间状态、报价模式、抬头信息、水电复核和人工补项能恢复。

## V1.1 候选项

试用反馈收集后，只把“必须修”和最高频“近期优化”放入 V1.1。当前候选：

- 修复或规避真实试用中发现的解析/报价/Excel 阻塞问题。
- 已完成：精简工程量表默认展示，让设计师先看到关键校对项。
- 已完成：优化健康检查文案，隐藏默认水电复核提醒。
- 已完成：微调前端信息层级和视觉样式，形成 V1.1 前端试用版。

## 暂不进入 V1.1

以下事项先记录，不作为 V1.1 的默认范围：

- DWG 直接上传。
- 设计师注册/登录。
- 客户档案和方案版本管理。
- 管理层后台和全局报价规则管理。
- 合同、收款、施工交底等扩展模块。
- 前端整体视觉大改版。
