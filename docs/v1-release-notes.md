# V1 可试用版本发布说明

当前稳定标签：`v1.0.1-real-drawing-quote-stable`

上一稳定标签：`v1-usable-dxf-quote-20260712`

发布日期：2026-07-13

关键修复提交：`b9b08b6 fix: filter bathroom rows for shower install export`

## 发布定位

这是 CAD 报价工具的第一个内部可试用稳定版本。V1 聚焦“规范方案文件上传后生成可校对、可追溯、可导出的报价草稿”，用于设计师和报价员先跑通真实项目流程。

V1 不追求完整平台化。登录注册、客户档案、后台管理、合同等能力放到后续版本；DWG 上传已在 V1.1 补入基础能力，但依赖服务器转换器。

## 本版已支持

- 上传规范 DXF/DWG 并解析空间、墙线、门窗、洞口、吊顶、柜体、洁具和外墙轮廓；DWG 会在服务端转为临时 DXF 后进入同一解析流程。
- 在“方案完整性复核”中查看关键总量和图形复核图，帮助设计师发现墙体、门窗、空间边界等漏画问题。
- 在空间工程量校对表中复核空间类型、状态、窗高、门窗扣减、窗帘墙宽等关键数据。
- 生成算量健康检查，并导出 CAD 修图清单。
- 生成报价映射 JSON。
- 导出三种 Excel 草稿：
  - 硬装
  - 整装
  - 硬装 + 自选增项
- Excel 草稿包含报价抬头、公共大项分组、空间章节、小计、直接费 `SUMIF` 合计公式、管理费/税金/总价公式和签名栏。
- 保存和导入校对快照，恢复工程量状态、报价模式、抬头信息、水电复核和人工补项。

## 已知限制

- DWG 上传依赖服务器安装并配置 ODA File Converter；若缺少转换器，系统会提示先安装转换器或从 CAD 另存/导出 DXF 后上传。
- 水电点位为系统推荐，不是真实水电施工图实测；正式报价前应人工复核。
- 客户、设计师、报价员等信息当前在页面报价抬头中填写并随快照保存，暂未接入账号和客户档案。
- 后台报价规则管理、设计师管理、客户方案版本管理暂未实现。
- 前端界面可用但仍偏工作台形态，整体视觉和交互美化放在后续版本。

## 推荐试用流程

1. 设计师按 `docs/cad-quote-drawing-spec-v1.md` 绘制并导出 DXF 或 DWG。
2. 上传方案文件，先看“方案完整性复核”和健康检查。
3. 修复 warning；info 类提醒至少人工看一遍。
4. 校对空间工程量和水电点位。
5. 填写报价抬头。
6. 导出 Excel 草稿。
7. 打开 Excel 做最终人工核价和打印预览。
8. 导出校对快照，保存为该客户方案的阶段记录。

## 后续优先级

1. 让 1-2 个设计师用真实项目试跑完整流程，记录卡点。
2. 使用 `docs/v1-trial-feedback.md` 汇总必须修、近期优化和后续版本事项。
3. 根据试用反馈微调前端信息层级。
4. 按 `docs/v2-upgrade-roadmap.md` 规划登录、客户档案、后台管理和 DWG 转换部署深化等平台化能力。
