# V1 设计师使用说明

更新时间：2026-07-13

本说明面向内部设计师，用于当前 V1 版本的日常试用。V1 暂时不包含账号体系和客户档案后台，客户信息先在页面的报价抬头区域填写，并随校对快照保存。

## 1. 上传方案文件

1. 打开前端：

   ```text
   http://127.0.0.1:3010/
   ```

2. 上传按规范绘制的 DXF 或 DWG 平面方案。
3. 等待系统生成空间工程量、方案完整性复核图和健康检查。

V1.1 支持 DXF/DWG 上传。DXF 会直接解析；DWG 会在服务端转为临时 DXF 后解析。若页面提示服务器缺少 ODA File Converter，可先从 CAD 另存或导出 DXF 后上传。

## 2. 看方案完整性复核

先看“方案完整性复核”区域。

重点检查：

- 空间数量是否符合方案。
- 建筑面积、地面面积是否和方案大致一致。
- 墙线长度是否异常偏小，偏小通常说明墙体线漏画。
- 门窗是否落在正确空间。
- 吊顶、柜体、洞口是否按规范图层表达。

默认图层只展示常用信息。需要看外墙、洞口、洁具、水电推荐点位时，点击“更多图层”。

## 3. 看健康检查

健康检查分两类：

- 需优先处理：通常会影响报价或空间归类，正式报价前应处理。
- 提醒：通常是需要设计师复核的信息，不一定阻止导出。

如果出现混合命名，例如 `客厅/电梯井`、`过道/电梯井`，建议回 CAD 拆分空间，不要把可计价空间和井道/洞口画在同一个 `QUOTE_ROOM` 里。

## 4. 校对空间工程量

空间工程量校对表用于确认每个空间的关键结果。

重点看：

- 空间名称和空间类型是否正确。
- 地面面积、墙线长度、窗洞面积是否明显异常。
- 厨房、卫生间、阳台等湿区的墙砖和防水是否合理。
- 门洞、窗洞扣减是否符合报价习惯。
- 窗帘墙宽候选是否需要人工调整。

校对完成后，可以把空间标为“已确认”。不计价空间可以标为“不计价”。

## 5. 复核水电点位

V1 的水电点位是系统推荐，不是真实水电施工图实测。

导出正式报价前建议：

- 打开水电点位复核区。
- 检查强电插座、开关、灯位、给水点、热水点、排水点数量。
- 必要时人工调整数量。
- 确认后再导出报价。

## 6. 填写报价抬头

在“报价抬头信息”区域填写：

- 地址名称
- 客户姓名
- 设计师
- 报价员
- 报价日期

这些信息会进入 Excel 表头和表尾。校对快照会保存这些信息，后续继续调整同一个客户方案时可以导入快照恢复。

## 7. 选择报价模式并导出

当前有三种报价模式：

- 硬装：只输出基础施工和硬装相关项目。
- 整装：输出当前已接入的全部项目。
- 硬装 + 自选增项：在硬装基础上勾选主材、定制、室内门、卫浴灯饰、窗帘美缝窗台石等增项。

导出 Excel 后，设计师可以继续在 Excel 中调整数量、单价和备注。直接费合计使用 `SUMIF` 汇总“小 计”，人工调整后可自动联动。

## 8. 保存校对快照

完成一次校对后，建议导出校对快照 JSON。

快照会保存：

- 当前工程量行和空间状态。
- 健康检查接受状态。
- 报价模式和自选增项选择。
- 报价抬头信息。
- 水电点位复核结果。
- Excel 人工补项数量。

后续客户方案调整时，重新上传 DXF/DWG 后可导入快照继续校对。

## 9. 记录试用反馈

V1 小范围试用时，请把卡住的地方记录到 `docs/v1-trial-feedback.md`。记录时优先写清楚：

- 是哪个项目和方案文件。
- 发生在哪个环节：上传、方案完整性复核、健康检查、工程量校对、水电复核、Excel 导出或快照恢复。
- 当前现象和期望结果。
- 是否影响金额或正式报价。

## 10. 对外报价前检查

正式发给客户前，至少确认：

- 健康检查没有未处理的 warning。
- 水电点位已经复核。
- 多楼层卫生间、盥洗区工程只在实际有卫生间的楼层生成；若发现异常楼层，应先检查空间类型和校对快照。
- 报价抬头信息完整。
- Excel 中强弱电工程、给排水工程、室内门、卫浴、吊顶、主材、定制等关键大项没有漏项。
- 打印预览为 A4，表格纵向居中，文字没有明显遮挡。
