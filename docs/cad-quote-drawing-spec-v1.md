# CAD 报价算量出图规范 v1

## 目标

本规范用于商品房装修报价的 DXF 自动算量验证。设计师在正常 CAD 平面方案基础上增加报价辅助图层，系统读取 DXF 后生成可校对的空间工程量依据表。

第一期只验证工程量准确性，不要求直接生成最终报价。

## 单位

- CAD 图纸单位统一为 `mm`。
- 系统输出面积单位为 `m2`，长度单位为 `m`。
- 如图纸单位不是 `mm`，上传时必须选择换算比例。

## 必需图层

| 图层名 | 用途 | 是否必需 |
|---|---|---|
| `QUOTE_ROOM` | 空间闭合边界，用于地面面积、顶面面积、空间归属 | 必需 |
| `QUOTE_WALL` | 实际墙体或可施工墙面几何，用于墙面计量长度 | 必需 |
| `QUOTE_OPENING` | 开放边界或非墙体边界，避免开放空间误计墙面 | 推荐 |
| `QUOTE_WINDOW` | 窗户宽度标记，用于窗洞扣减、窗台石、窗帘箱 | 推荐 |
| `QUOTE_DOOR` | 门洞宽度标记，默认不扣墙面 | 推荐 |
| `QUOTE_FLOOR` | 楼层标记，用于多楼层、别墅项目 | 可选 |
| `QUOTE_HEIGHT` | 空间级层高标记，优先于默认层高 | 可选 |
| `QUOTE_EXT_WALL` | 外墙计量线，第二阶段用于外墙面积 | 可选 |
| `QUOTE_TEXT` | 空间名称文字；也可读取普通文字 | 可选 |

## 空间边界

每个需要报价的空间必须在 `QUOTE_ROOM` 图层画一条闭合多段线。

要求：

- 一个空间对应一条闭合多段线。
- 多段线必须闭合，不能断开。
- 空间边界不能互相重叠。
- 客厅、餐厅、过道如果报价分开，就分别画闭合边界；如果合并报价，就画一个合并边界。
- `QUOTE_ROOM` 只用于空间范围、地面面积、顶面面积，不直接作为墙面周长。

## 墙面计量

`QUOTE_WALL` 表示真实墙体或可施工墙面几何。

系统计算墙面面积时使用：

```text
墙面计量长度 = 与空间关联的 QUOTE_WALL 长度
墙面展开面积 = 墙面计量长度 * 层高
乳胶漆面积 = 墙面展开面积 - 窗洞面积
```

门洞默认不扣，窗洞默认扣减。

## 开放边界

`QUOTE_OPENING` 用于客厅、餐厅、过道等开放空间。`QUOTE_ROOM` 为了计算地面面积必须闭合，但闭合边界中的开放边不是真实墙面，不能计入墙面项目。

规则：

- 被 `QUOTE_OPENING` 标注的边界明确不计墙面。
- 如果 `QUOTE_ROOM` 边界附近匹配到 `QUOTE_WALL`，则以 `QUOTE_WALL` 作为墙面计量依据。
- 系统不直接使用 `QUOTE_ROOM` 周长作为墙面施工周长。

## 空间名称

每个空间内部必须有一个文字名称。

建议格式：

```text
一层-客厅
一层-餐厅
一层-厨房
一层-卫生间
一层-主卧
一层-次卧
一层-书房
一层-阳台
```

要求：

- 名称文字放在对应 `QUOTE_ROOM` 闭合边界内部。
- 一个空间最好只有一个主名称。
- 名称不要放在墙体线或两个空间交界线上。
- 楼层和空间名尽量统一写法。

## 门窗

第一期门窗以线段表达宽度。

窗户：

- 放在 `QUOTE_WINDOW` 图层。
- 线段长度等于窗宽。
- 窗线尽量贴近对应空间边界或墙线。
- 窗高采用默认值，可在校对表修改。

门洞：

- 放在 `QUOTE_DOOR` 图层。
- 线段长度等于门洞宽。
- 门洞默认不扣墙面面积。
- 门高采用默认值，可在校对表修改。

门窗高度优先级：

```text
自身标注高度 > 空间类型默认高度 > 项目默认高度
```

## 层高

层高优先级：

```text
空间级 QUOTE_HEIGHT > 楼层默认层高 > 项目默认层高
```

第一期上传时填写项目默认层高，校对表中允许逐空间修改。`QUOTE_HEIGHT` 作为可选增强能力。

## 特殊空间

| 空间类型 | 第一阶段处理 |
|---|---|
| 楼梯 | 墙顶面可自动算；踏步、踢面、栏杆先手动输入或后续标注 |
| 楼梯过道 | 按普通过道计算 |
| 阳台 | 纳入第一期，计算地面、防水、墙面、窗相关数据 |
| 露台 | 预留类型，第一期可识别，复杂防水和外墙规则后续扩展 |
| 电梯井/设备井/管井 | 可识别，但默认不计价、不生成报价项目 |
| 外墙 | 使用 `QUOTE_EXT_WALL`，第一期只预留或输出长度，不作为核心准确性指标 |

## 异常提示

系统必须提示以下异常：

| 异常 | 处理 |
|---|---|
| 空间线未闭合 | 不计算，提示修图 |
| 空间没有名称 | 标记为未命名空间 |
| 名称不在任何空间内 | 提示确认归属 |
| 空间边界重叠 | 提示检查 |
| 墙线无法归属空间 | 放入未归属墙线列表 |
| 窗户无法归属空间 | 放入未归属窗户列表 |
| 单位异常 | 提示确认 DXF 单位 |
