# V1.1 前端试用版说明

更新时间：2026-07-14

V1.1 聚焦设计师正式试用前的前端信息层级和交互整理，不改变工程量公式、报价映射和 Excel 导出业务口径；本版补入 DWG 上传入口，DWG 会先由服务端转换为临时 DXF 后再复用现有解析流程。

## 本轮调整

- 系统名称改为“整装预算报价系统”，首屏只保留设计师主流程入口：方案上传、预算导出、培训 PPT 和规范资料。
- 校准 JSON、快照、报价映射 JSON、报价规则导入导出等内部工具统一收进“后台/校准工具”，默认不展示给设计师。
- “报价抬头信息”改为“方案信息”，增加确认按钮和已确认状态提示。
- 方案概览不再显示上传文件名，只显示空间数量、默认不计价数量、建筑面积和当前消息。
- “方案完整性复核”上移到方案信息之后，设计师上传方案后先看图形完整性。
- “Excel 可选补项”改为“方案报价可选项”，铝合金窗按面积、可编辑单价、总价和是否加入预算展示；单价默认沿用报价规则，本次修改会写入预算导出和校对快照。
- 健康检查不再显示默认的“水电点位待复核”提醒，只保留真正需要处理的风险或异常。
- 水电点位复核先显示总量和空间类别摘要，空间点位明细默认折叠；修改数量后按钮显示“确认修改”，确认后同步到后续预算导出。
- 空间工程量摘要区默认折叠，展开后每个空间也默认折叠；普通空间不要求确认，只有异常空间或设计师本次修改过的空间才展开并显示确认动作。空间卡片默认显示墙砖面积，不再显示墙线长度。
- 顶部“预算导出”保持为单纯按钮，当前预算模式作为按钮旁提示显示，例如“当前预算：硬装（半包）”，避免导出前选错模式。
- 方案上传入口支持 `.dxf` 和 `.dwg`；若服务器未配置 ODA File Converter，DWG 上传会给出可操作提示，设计师可先导出 DXF。
- 顶部“培训PPT”和“规范资料”会先打开 `/training` 资料中心，设计师可先浏览内容，再在页面内下载 PPT 或 Word 版规范；PPT 文件为 `apps/web/public/training/designer-training-v1.1.pptx`，如需更新课件，运行 `python tools/build_designer_training_ppt.py` 重新生成。
