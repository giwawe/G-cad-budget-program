# V1.1 前端试用版说明

更新时间：2026-07-13

V1.1 聚焦设计师正式试用前的前端信息层级和交互整理，不改变 DXF 解析、工程量公式、报价映射和 Excel 导出业务口径。

## 本轮调整

- 系统名称改为“全友整装预算报价系统”，首屏只保留设计师主流程入口：方案上传、预算导出、空间命名规范、画图规范。
- 校准 JSON、快照、报价映射 JSON、报价规则导入导出等内部工具统一收进“后台/校准工具”，默认不展示给设计师。
- “报价抬头信息”改为“方案信息”，增加确认按钮和已确认状态提示。
- 方案概览不再显示上传文件名，只显示空间数量、默认不计价数量、建筑面积和当前消息。
- “方案完整性复核”上移到方案信息之后，设计师上传方案后先看图形完整性。
- “Excel 可选补项”改为“方案报价可选项”，铝合金窗按面积、单价、总价和是否加入预算展示。
- 健康检查不再显示默认的“水电点位待复核”提醒，只保留真正需要处理的风险或异常。
- 水电点位复核的空间明细默认折叠，修改数量后通过确认按钮同步到后续预算导出。
- 空间工程量默认改成摘要卡片，保留计价类型、状态和关键工程量；完整明细表仍可展开查看。
